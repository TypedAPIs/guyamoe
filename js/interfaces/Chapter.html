groups

groups: {}
  • Defined in series.ts:32

The pages released by each group for the chapter. The key is one of the group IDs from Series.groups, and the value is the ordered list of page filenames.

A full page URL can be constructed using Series.slug, Chapter.folder, the group ID, and the page filename, by filling the template: {baseUrl}/media/manga/{slug}/chapters/{folder}/{groupID}/{pageName}

For example, if my base URL is https://guya.moe, my slug is Kaguya-Wants-To-Be-Confessed-To, my folder is 0259_0ixv0umx, my group ID is 7, and the page name is 01.png?v3, I would point a request to https://guya.moe/media/manga/Kaguya-Wants-To-Be-Confessed-To/chapters/0259_0ixv0umx/7/01.png?v3.

A list of the group IDs to prioritize when multiple groups upload the current chapter. This key may be omitted, in which case Series.preferred_sort should be considered, otherwise follow this array for the current chapter.
